# Messenger UI – Sprint 1

Учебный проект-мессенджер (1 спринт Яндекс.Практикума).

В этом спринте реализованы основные экраны интерфейса: авторизация, регистрация, список чатов, страница чата, профиль, редактирование профиля, смена пароля, а также страницы ошибок 404 и 5xx.  
Проект собран с помощью Vite, использует шаблонизатор Handlebars и SCSS-препроцессор.

---

## Деплой

Проект развёрнут на Netlify и доступен по адресу:

**https://astonishing-alfajores-0e4aef.netlify.app**

Автодеплой настроен из ветки `deploy` репозитория GitHub.

---

## Дизайн / прототипы

Прототипы экранов (авторизация, регистрация, профиль, редактирование профиля, смена пароля, 404 и 5xx) находятся в папке:

`/ui`

---

## Технологии

- **TypeScript**
- **Vite**
- **Handlebars** (шаблонизатор)
- **SCSS (Sass)** — препроцессор для стилей
- **Netlify** — деплой и автодеплой из ветки `deploy`

---

## Скрипты

Все команды выполняются из корня проекта:

```bash
# Запуск dev-сервера на http://localhost:3000
npm run start

# Альтернативный запуск dev-сервера
npm run dev

# Сборка проекта (TypeScript + Vite build)
npm run build

# Предпросмотр собранной версии
npm run preview
```
